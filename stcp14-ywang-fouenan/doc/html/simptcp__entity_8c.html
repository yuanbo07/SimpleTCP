int demultiplex_packet ( char *  buffer,
struct sockaddr_in *  udp_remote 
)

implemente la fonction de demultiplexage de SimpTCP declenchee a l'arrivee d'un PDU SimpTCP. A partir d'un PDU simpTCP recu, permet de determiner le socket SimpTCP destinataire deux cas de figure a considerer : Cas1) PDU destine a un "listening simpTCP socket" (cote serveur) suppose recevoir les PDU SimpTCP-SYN de demande d'etablissement d'une nouvelle connexion Cas 2) PDU destine a un "non listening socket" (socket cote client ou cote serveur cree suite a l'acceptation d'une demande de connexion

Paramètres:
bufferqui pointe sur le PDU SimpTCP (charge utile du paquet UDP recu)
udp_remotequi pointe sur l'adresse du socket UDP emetteur du PDU SimpTCP
Renvoie:
le descripteur du socket SimpTCP ou -1 s'il n'est destine a socket SimpTCP
